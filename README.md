Getting Started

1. Clone the Repository
First, clone the repository to your local machine:

git clone https://github.com/AbhinavKaintura/Real-Estate-Marketplace.git
cd your-repository-name

3. Install Dependencies
Next, install the required dependencies for the project:

Using npm:

npm install


3. Running the Development Server
Now you're ready to run the app locally! Use the following command to start the development server:

Using npm:
npm run dev
